AM numeric
==========

Collection of numeric facilities for C++


Classes
  - safe angle  	
  - dual number
  - split-complex number
  - quaternion  
  - ordinary biquaternion
  - split-biquaternion
  - dual quaternion (study biquaternion)
  
Other
  - number concept checking
  - checking for narrowing
  - some tests

Prerequisites
  - requires C++11 conforming compiler
  - tested with g++ 4.7.2
